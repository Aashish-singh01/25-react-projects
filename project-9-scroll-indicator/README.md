# 📊 Custom Scroll Indicator (React Project)

## 📌 Overview

This project is a **Custom Scroll Indicator App** built using React.
It fetches data from an API and displays a **dynamic scroll progress bar** that updates as the user scrolls through the page.

---

## 🚀 Features

* 📡 Fetch data from API
* 📜 Dynamic scroll progress indicator
* ⚡ Real-time scroll tracking
* 🔄 Responsive UI updates
* ⚛️ Built using React Hooks (`useState`, `useEffect`)

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* CSS
* Fetch API

---

## 📁 Project Structure

```id="92ny7r"
src/
```

---

## 💡 How It Works

* Data is fetched from an API using `fetch()`
* Products are stored in state and rendered dynamically
* Scroll event listener tracks user scroll position
* Scroll percentage is calculated and applied to progress bar width
